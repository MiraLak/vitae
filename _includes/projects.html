Activity Recognition Application

The goal of the application is to recognize physical activity with measurement coming from an accelerometer sensor. First, the measurements are collected from the accelerometer, then data are stored in Cassandra as a timeseries model. Data are analysed with Spark using Spark Streaming and Spark MlLib; Technologies:Java/ Scala, Spark, Spark Streaming, Spark ML Lib, Android SDK, Spring Boot,Cassandra Sources are available in my github account
